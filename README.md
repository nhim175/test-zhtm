# Requirements

- npm installed
- bower installed
- coffee installed
- port 3000, 3001 are free

#How to run

- npm install
- gulp clean-build && gulp
- Run api server: cd /path/to/project/server && coffee index.coffee
- open in browser: http://localhost:3000
